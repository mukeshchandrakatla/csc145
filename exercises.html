Exercises

Please utilize the source code from the Programs section and the book to complete the following exercises:

Object-Oriented Design

Arrays

  • Download the NetBeans project, import the project into NetBeans, and complete the specified sections. Make sure to write your name at the top. To submit your assignment, ZIP your completed project file and submit it on Moodle.

  • Download the NetBeans project, import the project into NetBeans, and do the following:

    • Write a Java program that prompts the user to insert five elements into an array.
    • The program should use a loop to determine the minimum and maximum elements in the array.
    • The program should prompt the user to insert five elements into a separate array and determine if it is equal or not equal to the first array.

  • Download the NetBeans project, import the project into NetBeans, and do the following:

    • Fill an object array with 3 Circle objects and print out the contents of the displayShape method using a for-loop.
    • Fill an object array with one Circle, one Square, and one Rectangle object and print out the contents of the displayShape method using a for-loop.
